# Mean-Variance-Standard Deviation Calculator
## Project Overview
A Python statistical analysis tool that calculates mean, variance, and standard deviation for any dataset.  
This project demonstrates understanding of descriptive statistics and data handling using Python and NumPy, making it ideal for analyzing datasets quickly and efficiently.

## Tools & Libraries
- Python
- NumPy
- Pandas (optional for dataset handling)
- Matplotlib / Seaborn (optional for visualizations)

## Dataset
The project can be used with any numeric dataset (CSV, Excel, or Python lists).

## Key Features
- Calculates mean, variance, and standard deviation
- Supports datasets of any size
- Optional visualization of data distributions
- Simple command-line interface for easy use

## Results & Insights
- Provides quick statistical insights into datasets
- Helps identify trends and variability in data
- Can be used as a base for more advanced data analysis projects
